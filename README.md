# Crop Cultivation Guidance System for Farmers

A Java-based desktop application developed to help farmers choose suitable and profitable crops based on soil type, season, land area, budget, and irrigation method.

## Features
- Farmer Registration & Login
- Crop Recommendation System
- Profit & Investment Analysis
- Irrigation Method Selection
- User-Friendly Java Swing Interface
- MySQL Database Connectivity

## Technologies Used
- Java
- Core Java
- Java Swing
- JDBC
- MySQL
- Eclipse IDE

## Objective
To provide a simple digital solution that helps farmers make better crop cultivation and investment decisions.

## Modules
- Farmer Authentication
- Crop Advisory Form
- Crop Recommendation Engine
- Result & Profit Analysis

## Future Enhancements
- Weather Forecast Integration
- Real-time Market Price Analysis
- Mobile Application Support

## Output
The system recommends profitable crops and displays investment, watering method, and estimated profit details for better farming decisions. Happy Farming
